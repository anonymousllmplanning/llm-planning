# Closed-Model Extension Aggregates

This folder contains sanitized aggregate tables for Appendix L. The tables cover
the eleven API-dependent closed models (OpenAI, Gemini, and Claude families) and
the Claude-only subset used to extend the paper's closed-model analysis.

Included files are model-level or pool-level aggregates only. They do not contain
GAIA raw questions, final answers, attachments, submitted answers, per-sample
review rows, or human-review audit examples.
